# 🚀 FIKI.host
### *The Automated Hosting Engine for the FIKI Empire*

**FIKI.host** is a custom-built deployment dashboard. It allows for the instant launching of web projects directly to Netlify using a private API connection.

---

## ⚡ Key Features
* **One-Tap Deployment:** Instantly turn HTML code into a live URL.
* **Backend Integration:** Powers the digital infrastructure for *Couture by Fiki* and *Fiki Portal*.
* **Custom Engine:** Runs on a specialized Netlify Function to bypass manual site creation.

## 🛠️ Infrastructure Setup
To keep the engine running, the following environment variables must be active in the Netlify site configuration:

| Variable | Purpose |
| :--- | :--- |
| `NETLIFY_TOKEN` | Grants the dashboard permission to build sites. |
| `NETLIFY_SITE_ID` | Directs the code to the correct hosting bucket. |
